TASK MANAGER APP

A Task-manager app where you can create , read , edit and delete tasks. Made with node.js, express.js, mongoDB and mongoose.

<h1>Installation</h1>

  Run 'npm install' to install the dependencies
  
  Run 'npm start' to start the website
